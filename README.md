# ProyectoTFCPython: Evolución de las versiones de Python

En esta web podras consultar todo lo que necesites saber sobre las versiones de Python.

Esta Web es una plataforma de investigación diseñada para analizar la trayectoria técnica de Python. Explora desde sus fundamentos en la década de los 90 hasta su consolidación como el lenguaje líder en Ciencia de Datos e Inteligencia Artificial en la actualidad.

## Objetivos del Proyecto

- **Explorar los fundamentos teóricos de Python:** Profundizar en los principios de diseño que definen al lenguaje (Zen de Python, legibilidad y tipado dinámico).
- **Analizar sus versiones más importantes:** Documentar la evolución desde la serie 1.x hasta la actualidad identificando los hitos que marcaron cada cambio generacional.
- **Estudiar su arquitectura y funcionamiento interno:** Comprender cómo el intérprete gestiona la memoria (Garbage Collector), el uso de tablas hash y la compilación a bytecode.
- **Comparar Python con otros lenguajes relevantes:** Evaluar sus ventajas y desventajas competitivas frente a lenguajes como Java, JavaScript o C ++.
- **Comprender su impacto en el mundo profesional y empresarial:** Analizar cómo su versatilidad ha transformado sectores como la Ciencia de datos, la Inteligencia Artificial y el desarrollo web corporativo.

## Estructura de Navegación
La web se organiza en módulos independientes para facilitar el acceso a la información técnica:

1. **Inicio (`index.html`)**: Visión general, genealogía rápida e impacto en ODS (Salud, Educación, Industria y Clima).
2. **Python 1.x (`version1.html`)**: Orígenes, tipos de datos iniciales y manejo de excepciones.
3. **Python 2.x (`version2.html`)**: Madurez, unificación de tipos/clases y programación funcional.
4. **Python 3.x (`version3.html`)**: Estándar actual, limpieza de sintaxis, Unicode nativo y optimización de memoria.

## Tecnologías y Estándares
- **HTML5 & CSS3**: Estructura semántica y diseño visual unificado mediante una hoja de estilos global.
- **Visual Studio Code**: Entorno de desarrollo principal.
- **Gráficos y Diagramas**: Incorporación de esquemas técnicos de arquitectura y estadísticas de uso de mercado (TIOBE/Stack Overflow).

## Componentes Técnicos Investigados
* **Intérprete CPython:** Flujo de ejecución desde el código fuente hasta la CPU.
* **Gestión de Memoria:** Análisis del Recolector de Basura y el Global Interpreter Lock (GIL).
* **Modelos de Datos:** Evolución de las estructuras internas (listas, diccionarios y generadores).
